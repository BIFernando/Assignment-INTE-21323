Assignment-INTE-21323 — Task Management System

A web-based Task Management System built as a team project for INTE-21323.
This system allows users to create, assign, and track tasks efficiently.

Technologies Used

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MySQL
ORM: Sequelize
Authentication: JWT (JSON Web Tokens)
Password Encryption: bcryptjs
Environment Variables:dotenv
Dev Tools: Nodemon, Git, GitHub
